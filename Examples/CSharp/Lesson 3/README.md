# Lekcja 3

## Typy danych

Typy podstawowe:
- Liczby całkowite [https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/builtin-types/integral-numeric-types]:
    - byte	System.Byte
    - sbyte	System.SByte
    - int	System.Int32
    - uint	System.UInt32
    - nint	System.IntPtr
    - nuint	System.UIntPtr
    - long	System.Int64
    - ulong	System.UInt64
    - short	System.Int16
    - ushort 
- Liczby zmiennoprzecinkowe
    - decimal System.Decimal
    - double System.Double
    - float	System.Single
- Pozostałe typy wbudowane
    - bool	System.Boolean
    - char	System.Char
